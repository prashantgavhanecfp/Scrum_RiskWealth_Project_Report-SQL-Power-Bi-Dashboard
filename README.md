Scrum_RiskWealth_Project_Report- SQL+Power Bi Dashboard

This repository contains the Scrum RiskWealth Analytics project, a data-driven solution built using SQL, Power BI dashboards, and CSV datasets to help wealth managers analyze customer risk profiles, portfolio distribution, and investment behavior.

The project integrates structured customer, portfolio, risk, and transaction data into an analytical model and presents insights through an interactive Power BI dashboard and a structured analytical report.
